## Stack

Hand-coded HTML & CSS. No framework, no build step, no JavaScript dependencies, no analytics.

```
.
├── index.html      The whole page
├── styles.css      Every style, hand-written
└── README.md       This file
```

## Type

- Display: [Fraunces](https://fonts.google.com/specimen/Fraunces)
- Body: [Newsreader](https://fonts.google.com/specimen/Newsreader)
- Mono: [JetBrains Mono](https://www.jetbrains.com/lp/mono/)

## Palette

## Local dev

Open `index.html` in a browser. That's it.

```bash
# or, if you prefer a server:
python3 -m http.server 8000
```

## Deployment

Deployed via GitHub Pages from the `main` branch.
Settings → Pages → Source: `main` / `/` (root).
